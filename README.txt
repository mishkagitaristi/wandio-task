
# Instruction for Setting up Development Environment with Gulp

#Step 1
!Follow this Link 'https://nodejs.org/en/' and Download and Install Latest LTS Version

#Step 2
!Open Terminal or Git Bash(run on current directory)
!Type in Terminal(Git Bash) - 'npm install' to install all the devDependencies from package.json file
!Once Modules Installation is Finished Type in Terminal 'npm install --global gulp-cli' to Install Gulp Globally

#Step 3
!Type 'gulp' in Terminal
!Enjoy ¯\_(ツ)_/¯
